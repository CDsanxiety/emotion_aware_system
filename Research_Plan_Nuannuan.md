# 暖暖 (Nuannuan) —— 基于认知型大模型与多模态融合的智能家居情感陪护系统研究计划书

## 第一章 项目摘要
人口老龄化与独居社会的到来使得情感孤独与心理健康问题成为现代社会的隐性危机。本项目旨在开发一套名为“暖暖”的认知型智能家居陪护系统。不同于传统的基于指令触发的智能助手，本项目通过整合视觉面部特征分析、语音情感特征提取以及认知型大语言模型（LLM），构建了一个“感知-理解-决策-执行”的情感闭环系统。系统通过非侵入式手段实时感知用户的情绪状态，利用大模型的逻辑推理能力实现“察言观色”，并通过拟人化的语音交互、环境灯光调节和主动关怀动作，提供深层次的情感慰藉。本项目目前已完成全链路算法架构验证，具备向智慧养老、智能家居及特种陪护机器人平滑迁移的技术基础。

## 第二章 设计需求分析
### 2.1 宏观背景：老龄化与心理慰藉缺位
根据国家卫健委及相关统计，我国养老模式呈现“9073”格局，即约 90% 的老年人选择居家养老。然而，现有的智能家居产品大多侧重于“物理安全”（如 WiCare 解决的跌倒监测），而在“精神安全”和“心理慰藉”领域存在巨大空白。独居老人、异地打拼的青年群体面临着严重的“情感孤岛”现象，长期的情感压抑易导致抑郁及其他心理疾病。

### 2.2 市场现状与痛点分析
1. **交互生硬（智能“伪命题”）**：现有智能助手（如小爱、小度）仍停留在“指令-响应”模式，无法主动感知用户的负面情绪（如隐忍的难过、隐秘的焦虑）。
2. **缺乏逻辑一致性**：传统的聊天机器人缺乏长短期记忆，无法根据用户的历史情感经历提供个性化的共情服务。
3. **隐私与设备碎片化**：现有的情感计算方案多依赖云端视频流，存在隐私泄露风险；且算法与硬件强耦合，难以适配不同形态的终端。

### 2.3 政策支持与导向
本项目响应《“十四五”数字经济发展规划》及《关于加快发展数字家庭 提高居住品质的指导意见》，重点突破“适老化”智能交互技术，推动智能家居从“单品智能”向“全屋认知智能”演进。

## 第三章 系统特色与技术创新
### 3.1 核心特色
1. **“由表及里”的深度感知**：系统不单识别笑容或泪水，更通过视觉背景分析（如环境乱象、光线明暗）与语音语调微变化，推断用户的压力水平。
2. **认知型大脑 (Cognitive Brain)**：接入阿里云百炼 Qwen-Turbo 等大模型，通过结构化提示词工程实现“类人化”的情感共鸣推理。
3. **软硬解耦的 HAL 架构**：定义了标准化的动作枚举（Action Enum），使得算法逻辑可以无缝适配 ROS 机器人、ESP32 物联网终端或移动端 App。

### 3.2 关键技术创新
*   **多模态特征对齐技术**：利用视觉多标签分类与音频情感分析（SER）进行交叉验证，识别“言不由衷”的情境。
*   **长短期记忆机制 (RAG-based Memory)**：基于 ChromaDB 向量数据库记录用户的情感画像，实现“记得住用户喜好”的贴心陪伴。
*   **物理表情映射 (PAD Model)**：将 LLM 的文本输出映射为心理学 PAD（愉悦度-激活度-优势度）模型，驱动硬件产生相应的拟人化反馈。

## 第四章 系统实现方案
### 4.1 总体架构设计
系统分为四层：
1. **感知层 (Perception)**：
    *   **视觉路径**：基于 OpenCV 与 FER 框架，实现 25fps 的实时面部分析。
    *   **听觉路径**：基于 Google Speech API 实现实时 ASR，同步提取音色频率特征。
2. **认知层 (Cognition)**：
    *   **LLM 推理链**：设计严格的 JSON 约束输出，涵盖 `emotion_score`, `reasoning_path`, `action_priority`。
    *   **决策机制**：当感知到极度负面情绪时，自动触发“高优安抚”模式。
3. **执行层 (Action)**：
    *   **TTS 语音合成**：采用 Edge-TTS 提供高自然度的拟人发声。
    *   **IoT 联动**：通过 MQTT/Serial 协议控制灯光颜色、音乐播放、机器人肢体动作。
4. **存储层 (Memory)**：持久化存储用户的情感交互历史，构建用户专属的心理画像。

### 4.2 软硬件接入细节
*   **主控环境**：Python 3.10+, 集成 Autogen 多智能体协作框架。
*   **硬件接口**：预留串口通信协议，支持对 STM32（参考 WiCare 架构）等实时控制端的指令下发。

## 第五章 预期成果与社会效益
1. **技术产出**：一套高可靠性的多模态情感计算软件系统，支持私有化部署。
2. **社会价值**：通过主动关怀减少居家老人的孤独感，降低抑郁风险，提升生活质量。
3. **商业潜力**：可作为智慧养老社区、高端智能家居及车载助手的核心情感引擎。
